### Commands

- `go init <repo.>`
  Initializing (プロジェクトおよび`go.mod, go.sum`の生成)
- `go get <package>`
  非推奨 (deprecated) | 今後削除予定
- `go install <package>`
  バイナリのパッケージをグローバルにインストール
- `go get tidy`
  Repo.ごとの`go.mod`ファイルに Source にて`import`されているモジュールを自動追加/削除
